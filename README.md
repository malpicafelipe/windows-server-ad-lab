# Windows Server Active Directory Lab

This project demonstrates the configuration of a Windows Server environment using Active Directory Domain Services in a virtual machine.

## Overview
- Installed and configured Active Directory
- Created Organizational Units (OUs), users, and groups
- Implemented role-based access control
- Configured NTFS and share permissions
- Applied Group Policy (GPO) settings
- Tested user access and security policies

## Key Skills
- Active Directory
- Windows Server
- Access Control
- Group Policy (GPO)
- System Administration
